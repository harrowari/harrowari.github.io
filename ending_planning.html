This is Ending Planning

During my time as a Planning Support Officer I was frustrated by the manual checking of property site plans against national/regional constraints maps. I knew that this task could be automated by running intersection queries for the property polygons against the constraints layers.

This task can be easily completed using a simple python script but I wanted to scale it so that it can run a report on a large property portfolio. Furthermore, I wanted to make this tool accessible as a web application so that it can be used by people with less programming experience.

Skills used

  • Docker for creating a portable, self-sufficient development environment for my team.
  • Spatial data science packages for analysis (GDAL/OGR, Geopandas, Matplotlib, Pandas).
  • FastAPI/Flask for creating a web microservice.
